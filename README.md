# 코딩테스트 연습 리파지토리
취업 시 필요한 코딩테스트를 준비하고 개인 역량을 향상시키기 위한 코딩테스트 연습 리파지토리입니다.

사용 언어: Java 8, Java 11

## 알고리즘 간략하게 정리
### 점근적 표기
* 𝜣-표기법
    * f(n)의 비율로 증가하는 함수
* 𝜪-표기법
    * 기껏해야 f(n)의 비율로 증가하는 함수
* 𝜴-표기법
    * 적어도 f(n)의 비율로 증가하는 함수

### 정렬
* 선택 정렬
* 버블 정렬
* 삽입 정렬
* 병합 정렬
* 퀵 정렬
* 힙 정렬
* 기수 정렬
* 계수 정렬

### 검색 트리
* 이진 검색 트리
* 레드 블랙 트리
* B-트리

### DP

### 그래프 탐색
* 깊이 우선 탐색
* 너비 우선 탐색
* 최소 신장 트리
    * 프림
    * 크루스칼 알고리즘
* 위상 정렬
* 최단 경로
    * 다익스트라 알고리즘
    * 벨만-포드 알고리즘
